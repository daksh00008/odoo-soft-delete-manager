Soft Delete Manager

Manage soft delete functionality for Odoo models seamlessly.

This module enables administrators to configure soft delete functionality for selected Odoo models. Instead of permanently deleting records, they are marked as deleted and can be recovered or permanently removed using a wizard. Perfect for preventing accidental data loss and managing data retention policies.

Key Features

  • Enable soft delete for specific Odoo models via settings.
  • Add a "Recover Deleted" button to tree views of configured models.
  • Use a wizard to recover or permanently delete soft-deleted records.
  • Hide soft-deleted records from default views using domain filters.
  • Securely manage data with the ability to restore records when needed.

Screenshots

Configure Soft Delete for Models

Select the models for which you want to enable soft delete functionality in the Settings menu.

Recover Deleted Button in Tree View

A "Recover Deleted" button appears in the tree view of configured models, allowing you to access deleted records.

Wizard to Restore or Delete Records

Use the wizard to view soft-deleted records and either restore them or permanently delete them.
